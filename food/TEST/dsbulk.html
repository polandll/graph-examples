Introduction to the DSE Bulk Loader tool.

The DSE Bulk Loader tool dsbulk can be used to load and unload DSE data in either CSV or JSON format.

About this document

Welcome to the DSE Bulk Loader documentation. To ensure that you get the best experience in using this document, take a moment to look at the Tips for using DataStax documentation.

About DSE Bulk Loader - dsbulk

The Datastax Bulk Loader tool is designed to provide users with the ability to both load and unload large amounts of data into DSE efficiently and reliably. Using dsbulk, CSV or JSON files can be rapidly loaded or unloaded from DSE. CSV or JSON files from both relational database exports and original data may be inserted into the DSE transactional database.
